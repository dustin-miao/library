## Chmax

### Summary
For some $a$ and $b$, assigns $a = \max(a, b)$. Returns `true` if the assignment is successful, and `false` otherwise. 

### Parameters
- `T &a`: A reference.
- `T b`: A value.

### Return
- `true` if $a < b$'; `false` otherwise.
