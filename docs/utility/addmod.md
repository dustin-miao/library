## Addmod

### Summary

An addition operator under modulo. 

### Preconditions

- `T`'s addition, subtraction, and comparison operators are defined

### Parameters
- `T a`: A non-negative value less than `MOD`
- `T b`: A non-negative value less than `MOD`
- `T MOD`: : A positive value
